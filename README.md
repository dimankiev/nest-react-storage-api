## Installation

```bash
$ yarn install
```

## Running the app

Before starting the app
```bash
# precompile scripts, run before building app
$ yarn run prebuild
```

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
